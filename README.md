# CT-Free Deep Learning-Based Attenuation and Scatter Correction for PET Imaging

## overview

This repository presents a deep learning framework for generating attenuation- and scatter-corrected PET images (MAC) from non-corrected PET images (NAC).

The aim of this project is to eliminate the need for CT-based correction to improve quantitative accuracy while reducing radiation exposure and simplifying clinical workflows.

This project was developed as part of my Master's research in Bio-Radiation Engineering.


## Key Features

- CT-free attenuation and scatter correction for PET imaging
- Deep learning-based image-to-image translation framework
- Implementation using Python, PyTorch, and MONAI
